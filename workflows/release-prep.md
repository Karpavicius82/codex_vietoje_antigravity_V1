# Release Prep Workflow

## Tikslas

Paruosti pakeitimus saugiam isleidimui.

## Eiga

1. Patikrink git busena ir pakeitimu apimti.
2. Surask projekto testavimo, build ir lint komandas.
3. Paleisk aktualius patikrinimus.
4. Paruosk PR aprasa pagal `templates/pr-description.md`.
5. Pazymek migracijas, config pokycius, feature flag'us ir rollback rizikas.

## Baigta, kai

- Pakeitimu apimtis aiski.
- Patikrinimai paleisti arba aiskiai nurodyti blockeriai.
- PR/release santrauka parengta.
