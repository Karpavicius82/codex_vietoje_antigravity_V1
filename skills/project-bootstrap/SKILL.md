# Project Bootstrap Skill

Use this skill when connecting a new repository to Codex vietoje Antigravity.

## Steps

1. Inspect repository structure.
2. Detect package manager, framework, test commands, build commands, lint commands, and dev server command.
3. Create or update `project.codex.json` in the project root.
4. Add project-specific notes only when they are factual.
5. Run a non-destructive validation command if available.

## Output

- Detected stack.
- Generated `project.codex.json`.
- Recommended first workflow.
