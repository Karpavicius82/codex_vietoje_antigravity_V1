# Frontend QA Skill

Use this skill for visual or interaction changes in a local web app.

## Steps

1. Identify the dev server command and expected local URL.
2. Start the app if needed.
3. Inspect the changed screen on desktop and mobile widths.
4. Check layout stability, overflow, contrast, and text fit.
5. Verify key interactions affected by the change.
6. Capture notes or screenshots when the browser tool is available.

## Output

- URL tested.
- Viewports tested.
- Issues found or confirmation that none were visible.
