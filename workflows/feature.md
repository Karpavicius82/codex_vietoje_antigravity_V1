# Feature Workflow

## Tikslas

Igyvendinti nauja funkcija maziausiu saugiu pakeitimu rinkiniu.

## Eiga

1. Perskaityk susijusius failus ir egzistuojancius naudojimo pavyzdzius.
2. Patikrink, ar jau yra panasi logika, komponentai, API ar testai.
3. Suplanuok siaura pakeitima.
4. Igyvendink funkcija pagal vietinius projekto standartus.
5. Pridek arba atnaujink testus, jei funkcija turi elgsenos rizika.
6. Paleisk aktualius patikrinimus.
7. Pateik trumpa rezultatu santrauka.

## Baigta, kai

- Funkcija veikia pagal uzduoti.
- Nera nereikalingu refaktorizaciju.
- Testai arba rankinis patikrinimas dokumentuoti galutiniame atsakyme.
